# MOVIESTOWN

A polished Netflix-style movie browsing website powered by TMDB.

## Features

- Search movies worldwide with TMDB-backed results
- Prioritize Indian and Mollywood titles in discovery and search
- Browse poster-heavy movie cards with ratings and genres
- Open a detailed modal with director, runtime, country, and awards
- Save favorites locally in the browser
- Launch trailer searches directly from the detail view

## API

This app uses TMDB and ships with a configured API key for quick local use.

## Run

1. Open `index.html` in a browser.
2. Type a movie title in the search box.
3. Click a card to view full details.
4. Use the genre chips to filter the catalog.

## Stack

- HTML
- CSS
- JavaScript
- TMDB API
